Guía de Sembrado en Casa

Aloe Vera

Pasos para el cultivo

  1. Seleccionar una maceta con drenaje
  2. Usar tierra para cactus
  3. Plantar a 5cm de profundidad
  4. Regar moderadamente

Cuidados especiales

  • Luz solar indirecta
  • Riego cada 2-3 semanas
  • Temperatura: 13-27°C

Orquídeas

Pasos para el cultivo

  1. Usar maceta transparente especial para orquídeas
  2. Sustrato específico para orquídeas
  3. Mantener humedad ambiente

Cuidados especiales

  • Luz filtrada
  • Riego semanal por inmersión
  • Humedad: 50-70%

Cactus

Pasos para el cultivo

  1. Elegir maceta con buen drenaje
  2. Usar sustrato para cactus
  3. Plantar superficialmente
  4. Esperar una semana antes del primer riego

Cuidados especiales

  • Luz solar directa
  • Riego cada 2-4 semanas
  • Temperatura: 15-35°C

Helecho de Boston

Pasos para el cultivo

  1. Usar maceta con buen drenaje
  2. Preparar sustrato rico en materia orgánica
  3. Mantener húmedo pero no encharcado
  4. Ubicar en zona sombreada

Cuidados especiales

  • Luz indirecta brillante
  • Riego frecuente
  • Temperatura: 16-24°C

Philodendron

Pasos para el cultivo

  1. Elegir maceta mediana con drenaje
  2. Usar sustrato bien drenado
  3. Plantar a profundidad media
  4. Mantener humedad constante

Cuidados especiales

  • Luz indirecta brillante
  • Riego cuando la superficie esté seca
  • Temperatura: 18-30°C

Rosa

Pasos para el cultivo

  1. Seleccionar maceta grande con drenaje
  2. Usar tierra rica en nutrientes
  3. Plantar a profundidad adecuada
  4. Podar regularmente

Cuidados especiales

  • Luz solar directa 6 horas
  • Riego profundo 2-3 veces por semana
  • Temperatura: 15-28°C

Pothos

Pasos para el cultivo

  1. Elegir maceta con drenaje
  2. Usar sustrato universal
  3. Plantar esquejes o división
  4. Regar moderadamente

Cuidados especiales

  • Luz indirecta o artificial
  • Riego cuando el sustrato esté seco
  • Temperatura: 15-30°C

Zamioculca

Pasos para el cultivo

  1. Usar maceta con buen drenaje
  2. Preparar sustrato bien drenado
  3. Plantar a profundidad media
  4. Regar con moderación

Cuidados especiales

  • Luz indirecta o sombra
  • Riego cada 2-3 semanas
  • Temperatura: 15-28°C
